# Finance Tracker App

Приложение для учета финансов

## Возможности

- Добавление транзакций (доход / расход)
- Редактирование транзакций
- Удаление транзакций
- Выбор даты с валидацией
- Категории для доходов и расходов
- Добавление комментария
- Статистика (графики по категориям)
- История транзакций с сортировкой по дате
- Сохранение данных в localStorage

## Технологии

- Angular
- Nx
- Taiga UI
- RxJS
- TypeScript
- ESLint + Prettier

## Установка и запуск

```bash
    npm install
    nx serve finance-tracker
```

**Приложение откроется по адресу:**
`http://localhost:4200`

## Линтинг и форматирование

```bash
    npm run lint
    npm run format
```
